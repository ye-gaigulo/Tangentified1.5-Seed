# Tangentified1.5-Seed
AngularJS 1.5 Seed using the generic Tangent template

### Setup Dependencies
  1. [npm](https://nodejs.org)
  1. [jspm](http://jspm.io/)

### Getting Started
    $ npm install
    $ jspm install

### Gulp Tasks
    $ gulp build
    $ gulp serve:dev
    $ gulp serve:dist
